# Getting Started with React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### Prepare the project

```sh
cp .env.sample .env.local

npm i

# Setup pre-commit hooks
npm run prepare

# Run app on local environment
npm start
```
